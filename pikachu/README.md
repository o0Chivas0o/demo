### 画一只皮卡丘中的注意事项

1. 用到了一个`setTimeout`改写`setInterval`来添加速度控制，原理就是延迟递归。
2. JS代码中`code`变量中字符串改成其他动画,即可画出另外的任何东西。
3. 灵感来自于会动的简历。
4. 通过`<pre></pre>`标签可展示所有的代码。
